# 📺 YouTube Link Manager (유튜브 링크 관리자)

나만의 유튜브 학습 및 취미 영상을 카테고리별로 깔끔하게 관리하는 안드로이드 애플리케이션입니다.

---

## 🚀 주요 기능

### 1. 카테고리별 폴더 구조
- 영상을 단순히 나열하는 것이 아니라 **공부, 자기계발, 개발, 취미** 등 원하는 카테고리별로 분류하여 관리할 수 있습니다.
- 메인 화면에서 폴더 형태의 UI를 통해 직관적으로 접근 가능합니다.

### 2. 세련된 유튜브 다크 테마
- 유튜브 브랜드 아이덴티티를 살린 **톤 다운된 레드(Soft Red)**와 **다크 그레이** 색상을 조합하여 눈이 편안한 다크 모드를 제공합니다.
- 카드 뷰(CardView) 디자인을 적용하여 각 아이템의 가독성을 높였습니다.

### 3. 원클릭 유튜브 연결
- 저장된 영상 링크를 클릭하면 즉시 외부 유튜브 앱이나 브라우저로 연결되어 영상을 시청할 수 있습니다.
- `Intent.FLAG_ACTIVITY_NEW_TASK`를 사용하여 앱 간 전환이 매끄럽습니다.

### 4. 간편한 영상 및 카테고리 관리
- **영상 관리**: 목록에서 아이템을 **길게 누르면** 수정 및 삭제 화면으로 이동합니다.
- **카테고리 이름 변경 및 삭제**: 카테고리 내부의 영상 목록 화면 우측 상단 메뉴를 통해 카테고리 이름을 변경하거나 삭제할 수 있습니다.
- **카테고리 순서 변경**: 메인 화면에서 카테고리를 **길게 눌러 드래그**하여 원하는 순서로 자유롭게 배치할 수 있습니다.
- **카테고리 추가**: 영상을 저장하는 중에 새로운 카테고리를 즉석에서 추가할 수 있습니다.

---

## 🛠 기술 스택
- **Language**: Java
- **Database**: Room Persistence (로컬 DB를 통한 안정적인 데이터 보관)
- **UI Components**: RecyclerView, CardView, Material Design Components, Toolbar
- **Architecture**: AppCompat, Jetpack libraries

---

## 🎨 디자인 포인트
- **Primary Color**: `#E53935` (YouTube Soft Red)
- **Background**: `#121212` (Dark Theme)
- **Interaction**: 뒤로가기 버튼, 스피너 카테고리 선택, 롱클릭 관리 기능 등 사용자 편의 중심의 UX.
